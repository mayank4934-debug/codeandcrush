
Code & Crush — Full Project Summary

1)What It Is
An AI-powered Virtual StudyDate Companion app that gamifies Computer Science learning through dating-sim style interactions, AI-driven teaching, and emotional support. Available as a web app, fully responsive for mobile, tablet, and desktop.

2)Tech Stack
Layer	Technology
Frontend	React + TypeScript + Tailwind CSS
Backend	Motoko canister (ICP)
State	React Context API
AI	OpenAI GPT-4o / Claude Sonnet 4.5
Voice	Web Speech API + ElevenLabs
Compiler	Judge0 (20+ languages)
Auth	Email/password + JWT + 6-digit OTP
Deployment	Internet Computer (ICP canister)
